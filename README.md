## Mochi is a guy package manager for macOS, iOS, iPadOS, tvOS, watchOS, and visionOS; all running under Procursus's apt/dpkg implementation. 

---
 
### Supported Versions
| Platform | Version | Jailbreak Type |
|----------|---------|----------------|
| iOS      | 15+     | rootless       |
| iPadOS   | 15+     | rootless       |
| macOS    | 12+     |                |
| tvOS     | —       | tbd            |
| watchOS  | —       | tbd            |
| visionOS | -       | tbd            |
