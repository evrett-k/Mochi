## Mochi is a GUI package manager for macOS, iOS, iPadOS, tvOS, watchOS, and visionOS; all running under apt/dpkg. 

---

### Supported Versions
| Platform | Version | Jailbreak Type |
|----------|---------|----------------|
| iOS      | 15+     | rootless       |
| iPadOS   | 15+     | rootless       |
| macOS    | 12+     |                |
| tvOS     | 15+     | rootless       |
| watchOS  | -       | tbd            |
| visionOS | -       | tbd            |
